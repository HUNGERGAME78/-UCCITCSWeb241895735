Web Development Basics
HTML CSS JavaScript React APIs
HTML

HTML is the standard markup language for creating webpages.

It provides the structure of a webpage.

<h1>Hello World</h1>

HTML elements form the building blocks of web interfaces.

  • Tags
  • Attributes
<p>This is a paragraph.</p>
CSS

CSS describes the visual style of HTML elements.

You can control colors, layouts, and animations.

body { background: lightblue; }
  • Selectors
  • Properties

CSS helps create visually appealing websites.

JavaScript

JavaScript is a programming language used for web interactivity.

It can modify HTML and CSS dynamically.

console.log("Hello JS");

JS runs directly in the browser.

  • Variables
  • Functions
let x = 5;
React

React is a JavaScript library for building UI components.

It focuses on reusable components and a virtual DOM.

const element = <h1>Hello</h1>;

React apps are highly scalable.

  • Components
  • Hooks
useState()
APIs

APIs let apps communicate with each other.

They allow developers to access external data.

fetch("https://api.com")

Most modern apps rely on APIs.

  • REST
  • JSON
{ "name": "API" }
